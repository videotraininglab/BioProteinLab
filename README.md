# BioProteinLab Pro Kinetic Theme

Final-polished Shopify OS 2.0 Theme for BioProteinLab.

## Enthalten

- Scroll Morph Hero Pro im Stil moderner 21st.dev-Komponenten, umgesetzt in Shopify Liquid, CSS und Vanilla JavaScript ohne React-/Framer-Abhängigkeit.
- Nike-inspirierte Scroll-Zerlegung für Produkt- und Zutatenvisualisierung.
- Modern Rounded/Tinker-Style Karten mit großen Radien, Glasflächen, Hover Tilt und weichen Schatten.
- Vollständig aus der BioProteinLab Brand Guideline übernommen:
  - Farben: #0F5F4A, #A8C3A0, #E8DCC2, #F7F5EF, #2B3B2B
  - Typografie: Montserrat für Headlines/UI, Lora für Body/Zitate
  - Markenwerte: Bio-zertifizierte Qualität, hochbioverfügbare Proteine, nachhaltige Herstellung, Gesundheit & Performance, rein natürliche Inhaltsstoffe
  - Bildwelt: hell, clean, minimalistisch, natürlich, warm und hochwertig
  - Markenbotschaft: Natürlich. Hochwertig. Bio. Für deinen aktiven Lebensstil.
- Alle gelieferten Guideline-Bilder wurden als WebP-Webassets optimiert und zusätzlich in hochwertige Crops für Hero, Karten und Sections zerlegt.

## Import in Shopify

Shopify Admin → Online Store → Themes → Add theme → Upload zip.

## Nach dem Upload

1. Im Theme Editor echte Produkte und Collections verbinden.
2. Logo in den Theme Settings hochladen.
3. Produktbilder prüfen und bei Bedarf die Morph Hero Cards im Theme Editor austauschen.
4. Vor Livegang mobile Ansicht, Warenkorb, Produktseite und Checkout-Verlinkung testen.
